# Installation

## Requirements

* Go >=1.9

## Build

Build Apla:
```
go get github.com/AplaProject/go-apla
```

# Running

Create Apla directory and copy binary:
```
mkdir ~/apla
cp $GOPATH/bin/go-apla ~/apla
```

Run Apla:
```
~/apla/go-apla
```

To work through GUI you need to install https://github.com/AplaProject/apla-front
